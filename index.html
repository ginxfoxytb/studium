Connexion à Firebase…

🔧 Configuration Firebase requise

  1. Allez sur console.firebase.google.com → créez un projet
  2. Menu gauche → Firestore Database → Créer → mode Test
  3. ⚙️ Paramètres du projet → Vos applications → icône </> Web
  4. Copiez l'objet firebaseConfig et collez-le dans le fichier HTML à l'endroit indiqué

Ouvrez le fichier studium.html dans un éditeur de texte (Notepad, VS Code…)
Cherchez la ligne apiKey: "VOTRE_API_KEY" et remplacez les 6 valeurs par celles de votre projet Firebase.

const firebaseConfig = {
  apiKey: "AIzaSy…",
  authDomain: "mon-projet.firebaseapp.com",
  projectId: "mon-projet",
  storageBucket: "mon-projet.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123…"
};
